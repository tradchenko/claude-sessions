# Plan 04-04: Интеграционные тесты CLI-команд

## Objective
Покрыть все 5 CLI-команд (`list`, `resume`, `restore`, `install`, `extract-memory`) интеграционными тестами. Тесты работают с реальной файловой системой в temp directories, вызывают command functions напрямую (не через child_process), проверяют happy path и критичные edge cases.

## must_haves
- 5 тест-файлов в `tests/integration/` — по одному на команду
- Каждая команда покрыта: happy path + 2-3 edge cases (пустые данные, повреждённые файлы, отсутствующие агенты)
- E2E тест миграции: старый формат → `runMigrations()` → `list` работает с мигрированными данными
- Все тесты изолированы: temp dir + cleanup
- `npm test` подхватывает новые файлы автоматически (glob `tests/**/*.test.mjs`)
- Все тесты проходят в CI
