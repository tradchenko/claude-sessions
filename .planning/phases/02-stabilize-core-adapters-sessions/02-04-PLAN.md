# Plan 02-04: List — дедупликация, фильтры, пустое состояние

## Контекст

loader.ts concatenates результаты из всех адаптеров без дедупликации. Companion дублирует Claude-сессии. Пустое состояние показывает "Сессий не найдено" без подсказки. Фильтры в основном работают, но нужна проверка.
